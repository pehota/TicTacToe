# Tic Tac Toe

A Tic Tac Toe game built on [Ampersand.js](http://ampersandjs.com)

## How to run it

1. download/install [node.js](http://nodejs.org/)
1. install dependencies: `npm install`
1. run it: `npm start`
1. open http://localhost:3000 in a browser
